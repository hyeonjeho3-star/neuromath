# 6주차 - 등식의 성질과 일차방정식 기초
tags: 중1, 등식, 방정식, 이항

---

Q: 등식의 정의는?
A: 등호(=)를 사용하여 두 식이 같음을 나타낸 식

Q: 방정식의 정의는?
A: 미지수(문자)의 값에 따라 참이 되기도 하고 거짓이 되기도 하는 등식

Q: 항등식의 정의는?
A: 미지수에 어떤 값을 대입해도 항상 참인 등식

Q: x + 2 = 5는 방정식인가, 항등식인가?
A: 방정식 (x = 3일 때만 참)

Q: 2(x + 1) = 2x + 2는 방정식인가, 항등식인가?
A: 항등식 (모든 x에 대해 참)

Q: 방정식의 해(근)란?
A: 방정식을 참이 되게 하는 미지수의 값

Q: x + 3 = 7의 해는?
A: x = 4

Q: 좌변과 우변의 정의는?
A: 등호 왼쪽이 좌변, 오른쪽이 우변

Q: 양변이란?
A: 좌변과 우변을 합쳐 부르는 말

Q: 등식의 성질 1: a = b이면 a + c = ?
A: b + c (양변에 같은 수를 더해도 등식 성립)

Q: 등식의 성질 2: a = b이면 a - c = ?
A: b - c (양변에서 같은 수를 빼도 등식 성립)

Q: 등식의 성질 3: a = b이면 a × c = ?
A: b × c (양변에 같은 수를 곱해도 등식 성립)

Q: 등식의 성질 4: a = b이면 a ÷ c = ? (단, c ≠ 0)
A: b ÷ c (양변을 같은 수로 나눠도 등식 성립)

Q: 이항의 정의는?
A: 등식에서 한 항을 부호를 바꾸어 다른 변으로 옮기는 것

Q: x + 3 = 7에서 +3을 이항하면?
A: x = 7 - 3 = 4 (부호가 -로 바뀜)

Q: x - 5 = 2에서 -5를 이항하면?
A: x = 2 + 5 = 7 (부호가 +로 바뀜)

Q: 일차방정식의 정의는?
A: 미지수의 최고 차수가 1인 방정식 (ax + b = 0 형태, a≠0)

Q: 일차방정식 2x = 6의 해는?
A: x = 3 (양변을 2로 나눔)

Q: 일차방정식 x/3 = 4의 해는?
A: x = 12 (양변에 3을 곱함)

Q: 3x + 2 = 11의 해를 구하는 과정은?
A: 3x = 11 - 2 → 3x = 9 → x = 3

Q: 2x - 5 = x + 3의 해는?
A: x = 8 (2x - x = 3 + 5)

Q: 4x + 1 = 2x + 9의 해는?
A: x = 4 (4x - 2x = 9 - 1 → 2x = 8)

Q: 일차방정식을 푸는 순서는?
A: ① 괄호 풀기 ② 이항하여 동류항 정리 ③ ax = b 형태로 만들기 ④ 양변을 a로 나누기

Q: 5(x - 1) = 10의 해는?
A: x = 3 (5x - 5 = 10 → 5x = 15 → x = 3)

Q: 방정식의 해를 검산하는 방법은?
A: 구한 해를 원래 방정식에 대입하여 좌변 = 우변인지 확인

Q: 3x = 0의 해는?
A: x = 0

Q: 0x = 0의 해는?
A: 모든 수 (해가 무수히 많음, 부정)

Q: 0x = 5의 해는?
A: 해가 없음 (불능)
