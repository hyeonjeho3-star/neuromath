# 7주차 - 복잡한 일차방정식 풀이
tags: 중1, 일차방정식, 괄호, 분수, 소수

---

Q: 괄호가 있는 방정식을 풀 때 첫 번째 단계는?
A: 분배법칙을 이용하여 괄호를 풀기

Q: 2(x + 3) = 14의 해는?
A: x = 4 (2x + 6 = 14 → 2x = 8)

Q: 3(x - 2) = 2(x + 1)의 해는?
A: x = 8 (3x - 6 = 2x + 2 → x = 8)

Q: 4(2x - 1) = 3(x + 2) + 5의 해는?
A: x = 3 (8x - 4 = 3x + 6 + 5 → 5x = 15)

Q: -(x - 3) = 2의 해는?
A: x = 1 (-x + 3 = 2 → -x = -1)

Q: 소수가 있는 방정식을 풀 때 유용한 방법은?
A: 양변에 10, 100 등을 곱하여 정수로 만들기

Q: 0.2x + 0.3 = 0.5의 해는?
A: x = 1 (양변에 10을 곱하면 2x + 3 = 5)

Q: 0.3x - 0.1 = 0.2x + 0.4의 해는?
A: x = 5 (3x - 1 = 2x + 4 → x = 5)

Q: 0.5x + 1.2 = 0.3x + 2의 해는?
A: x = 4 (5x + 12 = 3x + 20 → 2x = 8)

Q: 0.02x + 0.1 = 0.06의 해는?
A: x = -2 (양변에 100을 곱하면 2x + 10 = 6)

Q: 분수가 있는 방정식을 풀 때 유용한 방법은?
A: 양변에 분모의 최소공배수를 곱하여 분수를 없앰

Q: x/2 = 3의 해는?
A: x = 6 (양변에 2를 곱함)

Q: x/3 + 1 = 2의 해는?
A: x = 3 (x/3 = 1 → x = 3)

Q: x/2 + x/3 = 5의 해는?
A: x = 6 (양변에 6을 곱하면 3x + 2x = 30)

Q: (x+1)/2 = (x-1)/3의 해는?
A: x = 5 (양변에 6을 곱하면 3(x+1) = 2(x-1) → 3x+3 = 2x-2)

Q: (2x-1)/3 = (x+2)/2의 해는?
A: x = 8 (양변에 6을 곱하면 2(2x-1) = 3(x+2))

Q: 2x/5 - 1 = x/3의 해는?
A: x = 15 (양변에 15를 곱하면 6x - 15 = 5x)

Q: 0.5(x - 2) = 1의 해는?
A: x = 4 (양변에 2를 곱하면 x - 2 = 2)

Q: (x+3)/4 + 1 = x/2의 해는?
A: x = 7 (양변에 4를 곱하면 x+3+4 = 2x)

Q: 일차방정식 2x + 5 = 2x + 3의 해는?
A: 해가 없다 (0x = -2는 불능)

Q: 일차방정식 3(x + 1) = 3x + 3의 해는?
A: 모든 수 (0x = 0은 부정)

Q: 해가 없는 방정식을 무엇이라 하는가?
A: 불능

Q: 해가 무수히 많은 방정식을 무엇이라 하는가?
A: 부정

Q: ax = b에서 a = 0, b ≠ 0이면?
A: 해가 없다 (불능)

Q: ax = b에서 a = 0, b = 0이면?
A: 해가 무수히 많다 (부정)

Q: 0.1(x + 2) = 0.3(x - 2)의 해는?
A: x = 4 (x + 2 = 3(x - 2) → x + 2 = 3x - 6)

Q: x/2 - (x-3)/4 = 1의 해는?
A: x = 1 (양변에 4를 곱하면 2x - (x-3) = 4)

Q: 1/3(x - 6) = 1/2(x - 4)의 해는?
A: x = 0 (양변에 6을 곱하면 2(x-6) = 3(x-4))

Q: 검산할 때 주의할 점은?
A: 구한 해를 원래 방정식에 대입하여 좌변 = 우변인지 반드시 확인
